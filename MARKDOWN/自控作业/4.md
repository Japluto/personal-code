### **第 6 题**

**思路分析**
1.  **目标**: 求解直流电机的额定负载转矩 $T_L$。
2.  **核心公式**:
    * 电枢电压平衡方程: $U_a = E_a + I_a R_a$
    * 感应电动势 (反电动势) 方程: $E_a = C_e n$
    * 电磁转矩方程: $T_{em} = C_T I_a$
    * 转矩平衡方程: $T_{em} = T_L + T_0$
3.  **解题步骤**:
    * 首先，利用额定电压、额定电流和电枢电阻，根据电压平衡方程求出额定运行时的感应电动势 $E_a$。
    * 然后，利用额定转速和刚求出的感应电动势，计算出电机的电动势常数 $C_e$。
    * 对于直流电机，在统一单位制下，转矩常数 $C_T$ 在数值上等于电动势常数 $C_e$。
    * 利用转矩常数 $C_T$ 和额定电流 $I_a$ 计算出电磁转矩 $T_{em}$。
    * 最后，根据转矩平衡方程，用电磁转矩减去已知的空载转矩 $T_0$，得到额定负载转矩 $T_L$。

**解答过程**

1.  **求额定运行时的感应电动势 $E_a$**:
    根据电压平衡方程 $U_a = E_a + I_a R_a$，可得：
    $E_a = U_a - I_a R_a = 110 \, \text{V} - 0.4 \, \text{A} \times 50 \, \Omega = 110 \, \text{V} - 20 \, \text{V} = 90 \, \text{V}$

2.  **求电动势常数 $C_e$**:
    根据感应电动势方程 $E_a = C_e n$，可得：
    $C_e = \frac{E_a}{n} = \frac{90 \, \text{V}}{3600 \, \text{r/min}} = 0.025 \, \text{V/(r/min)}$

3.  **求电磁转矩 $T_{em}$**:
    转矩常数 $C_T$ 与电动势常数 $C_e$ 的关系为 $C_T = 9.55 C_e$（当 $n$ 的单位为 r/min 时），或者我们可以先将 $C_e$ 转换为国际单位制下的 $K_e$，再利用 $K_T = K_e$。这里使用前者更方便：
    $C_T = 9.55 \times C_e = 9.55 \times 0.025 = 0.23875 \, \text{N} \cdot \text{m/A}$
    根据电磁转矩方程 $T_{em} = C_T I_a$，可得：
    $T_{em} = 0.23875 \, \text{N} \cdot \text{m/A} \times 0.4 \, \text{A} = 0.0955 \, \text{N} \cdot \text{m}$

4.  **求额定负载转矩 $T_L$**:
    根据转矩平衡方程 $T_{em} = T_L + T_0$，可得：
    $T_L = T_{em} - T_0 = 0.0955 \, \text{N} \cdot \text{m} - 0.015 \, \text{N} \cdot \text{m} = 0.0805 \, \text{N} \cdot \text{m}$

**最终答案**: 该电机的额定负载转矩是 **0.0805 N·m**。

### **第 7 题**

**思路分析**
这是一个同轴连接的电动-发电机组问题，两台电机完全相同，励磁也相同，意味着它们的电机常数 $C_e$ 和 $C_T$ 是相等的。

* **第1问**: 求发电机空载时的电枢电压。
    1.  首先分析电动机在空载（指发电机不接负载）时的状态。利用电动机的电压、电流、电阻和转速，可以求出其感应电动势 $E_m$，进而求出电机常数 $C_e$。
    2.  发电机与电动机同轴，转速相同。
    3.  发电机的感应电动势 $E_g$ 可以通过 $E_g = C_e n$ 计算得出。
    4.  空载时，发电机电枢电流为零，因此其端电压等于其感应电动势。

* **第2问**: 求发电机接上负载后，机组的转速。
    1.  这是一个新的稳态，转速 $n$ 是未知的。
    2.  **电动机侧**: 其感应电动势 $E_m' = C_e n$。其电枢电流 $I_m' = (U_m - E_m') / R_a$。其输出的电磁转矩 $T_{em,m}' = C_T I_m'$。
    3.  **发电机侧**: 其感应电动势 $E_g' = C_e n$。负载电阻 $R_L = 0.5 \, \text{k}\Omega = 500 \, \Omega$。发电机回路的总电阻为 $R_a + R_L$。其电枢电流 $I_g' = E_g' / (R_a + R_L)$。它产生的电磁转矩（作为负载）为 $T_{em,g}' = C_T I_g'$。
    4.  **系统转矩平衡**: 电动机产生的电磁转矩需要克服两部分：一部分是机组的空载（摩擦）转矩 $T_0$，另一部分是发电机带负载后产生的电磁阻力转矩 $T_{em,g}'$。
    5.  空载转矩 $T_0$ 可由初始空载状态下的电动机电流计算得出。
    6.  建立转矩平衡方程 $T_{em,m}' = T_0 + T_{em,g}'$，代入所有表达式，即可解出唯一的未知数——转速 $n$。

**解答过程**

**1) 发电机空载时的电枢电压为多少？**
1.  **求电机常数 $C_e$**:
    在发电机空载时，电动机的感应电动势为：
    $E_m = U_m - I_m R_a = 110 \, \text{V} - 0.12 \, \text{A} \times 75 \, \Omega = 110 \, \text{V} - 9 \, \text{V} = 101 \, \text{V}$
    电机常数为：
    $C_e = \frac{E_m}{n} = \frac{101 \, \text{V}}{4500 \, \text{r/min}} \approx 0.02244 \, \text{V/(r/min)}$

2.  **求发电机空载感应电动势 $E_g$**:
    发电机与电动机同轴，转速相同，电机特性也相同，所以：
    $E_g = C_e n = 0.02244 \, \text{V/(r/min)} \times 4500 \, \text{r/min} = 101 \, \text{V}$

3.  **求发电机空载电枢电压 $U_{g0}$**:
    空载时，发电机电枢电流 $I_g = 0$，所以端电压等于感应电动势：
    $U_{g0} = E_g - I_g R_a = 101 \, \text{V} - 0 = 101 \, \text{V}$

**2) 电动机的电枢电压仍为110V，而发电机接上0.5kΩ的负载时，机组的转速 n 是多少？**
1.  **求空载转矩 $T_0$ 和转矩常数 $C_T$**:
    $C_T = 9.55 C_e = 9.55 \times 0.02244 \approx 0.2143 \, \text{N} \cdot \text{m/A}$
    机组的空载转矩由初始状态下电动机的电流提供：
    $T_0 = C_T I_m = 0.2143 \, \text{N} \cdot \text{m/A} \times 0.12 \, \text{A} \approx 0.0257 \, \text{N} \cdot \text{m}$

2.  **建立新稳态下的方程 (设新转速为 n')**:
    * 电动机电枢电流: $I_m' = \frac{U_m - E_m'}{R_a} = \frac{110 - C_e n'}{75}$
    * 电动机电磁转矩: $T_{em,m}' = C_T I_m' = 0.2143 \times \frac{110 - 0.02244 n'}{75}$
    * 发电机感应电动势: $E_g' = C_e n'$
    * 发电机电枢电流: $I_g' = \frac{E_g'}{R_a + R_L} = \frac{C_e n'}{75 + 500} = \frac{0.02244 n'}{575}$
    * 发电机电磁转矩: $T_{em,g}' = C_T I_g' = 0.2143 \times \frac{0.02244 n'}{575}$

3.  **建立并求解转矩平衡方程**:
    $T_{em,m}' = T_0 + T_{em,g}'$
    $0.2143 \times \frac{110 - 0.02244 n'}{75} = 0.0257 + 0.2143 \times \frac{0.02244 n'}{575}$
    为简化计算，注意到 $C_T$ 可以从等式两边含有它的项中约去（$T_0$ 项除外），我们可以直接用电流关系建立平衡：$I_m' = I_0 + I_g'$ （其中$I_0=0.12A$是产生空载转矩的等效电流）
    $\frac{110 - C_e n'}{R_a} = 0.12 + \frac{C_e n'}{R_a + R_L}$
    $\frac{110 - 0.02244 n'}{75} = 0.12 + \frac{0.02244 n'}{575}$
    $\frac{110}{75} - \frac{0.02244 n'}{75} = 0.12 + \frac{0.02244 n'}{575}$
    $1.4667 - 0.0002992 n' = 0.12 + 0.0000390 n'$
    $1.3467 = (0.0002992 + 0.0000390) n'$
    $1.3467 = 0.0003382 n'$
    $n' = \frac{1.3467}{0.0003382} \approx 3981.4 \, \text{r/min}$

**最终答案**:
1) 发电机空载时的电枢电压为 **101 V**。
2) 机组的转速约为 **3981 r/min**。

### **第 15 题**

**思路分析**
这是一个关于直流伺服电机特性的问题。
* **第1问**: 绘制机械特性。机械特性是指转速 $n$ 与（电磁）转矩 $T$ 之间的关系曲线。对于他励或永磁直流电机，其机械特性方程为 $n = \frac{U_a}{C_e} - \frac{R_a}{C_e C_T} T_{em}$。这是一条直线。我们可以根据题目给出的两个工作点 $(I_{a1}, n_1)$ 和 $(I_{a2}, n_2)$ 来确定这条直线。需要先计算出这两个点对应的电磁转矩。为此，需要先求出电机常数。
* **第2问**: 求机电时间常数。机电时间常数的公式为 $\tau_m = \frac{J_m R_a}{K_e K_T}$（其中 $K_e, K_T$ 为国际单位制下的电机常数）。因此，需要先求出 $R_a$ 和电机常数。
* **第3问**: 讨论电枢电压变化时，机电时间常数是否变化。分析 $\tau_m$ 的公式，看其中各项是否与电枢电压 $U_a$ 有关。

**解题过程**
题目中未给出电枢电阻 $R_a$，需要先通过两个工作点的数据联立求解。

1.  **求 $R_a$ 和 $C_e$**:
    在两个工作点，电压平衡方程成立：
    * 点1: $U_a = E_{a1} + I_{a1} R_a \implies 100 = C_e n_1 + 0.05 R_a \implies 100 = C_e \cdot 4500 + 0.05 R_a$
    * 点2: $U_a = E_{a2} + I_{a2} R_a \implies 100 = C_e n_2 + 0.2 R_a \implies 100 = C_e \cdot 3000 + 0.2 R_a$
    联立求解：
    $C_e \cdot 4500 + 0.05 R_a = C_e \cdot 3000 + 0.2 R_a$
    $1500 C_e = 0.15 R_a \implies R_a = 10000 C_e$
    代入第一个方程：
    $100 = 4500 C_e + 0.05 (10000 C_e) = 4500 C_e + 500 C_e = 5000 C_e$
    $C_e = \frac{100}{5000} = 0.02 \, \text{V/(r/min)}$
    $R_a = 10000 \times 0.02 = 200 \, \Omega$

2.  **绘制机械特性**:
    * 转矩常数: $C_T = 9.55 C_e = 9.55 \times 0.02 = 0.191 \, \text{N} \cdot \text{m/A}$
    * 计算两个工作点的电磁转矩：
        * $T_{em1} = C_T I_{a1} = 0.191 \times 0.05 = 0.00955 \, \text{N} \cdot \text{m}$ (此时转速 $n_1=4500$ r/min)
        * $T_{em2} = C_T I_{a2} = 0.191 \times 0.2 = 0.0382 \, \text{N} \cdot \text{m}$ (此时转速 $n_2=3000$ r/min)
    * 理想空载转速 (当 $T_{em}=0$ 时)：$n_0 = \frac{U_a}{C_e} = \frac{100 \, \text{V}}{0.02 \, \text{V/(r/min)}} = 5000 \, \text{r/min}$
    * 堵转转矩 (当 $n=0$ 时)：$T_{st} = \frac{C_T U_a}{R_a} = \frac{0.191 \times 100}{200} = 0.0955 \, \text{N} \cdot \text{m}$
    * 在坐标系中，以转矩为横轴，转速为纵轴，连接点 $(0, 5000)$ 和点 $(0.0955, 0)$ 即可得到机械特性曲线（一条直线）。

3.  **求机电时间常数 $\tau_m$**:
    * 首先将电机常数转换为国际单位制 (rad/s)：
        $K_e = \frac{E_a}{\omega} = C_e \times \frac{60}{2\pi} = 0.02 \times 9.55 \approx 0.191 \, \text{V/(rad/s)}$
        $K_T = K_e = 0.191 \, \text{N} \cdot \text{m/A}$
    * 计算时间常数：
        $\tau_m = \frac{J_m R_a}{K_e K_T} = \frac{(3.7 \times 10^{-6} \, \text{kg} \cdot \text{m}^2) \times 200 \, \Omega}{0.191 \times 0.191} \approx 0.0203 \, \text{s}$

4.  **讨论电压变化对时间常数的影响**:
    机电时间常数 $\tau_m = \frac{J_m R_a}{K_e K_T}$。公式中的转动惯量 $J_m$、电枢电阻 $R_a$、电机常数 $K_e$ 和 $K_T$ 都是电机的固有物理参数，它们不随电枢电压 $U_a$ 的变化而变化。因此，当电枢电压变为110V时，机电时间常数不变。

**最终答案**:
1) 机械特性是一条直线，其理想空载转速为 **5000 r/min**，堵转转矩为 **0.0955 N·m**。
2) 电机时间常数约为 **0.0203 s** (或 20.3 ms)。
3) 当电枢电压变为110V时，机电时间常数 **不变化**，因为它仅由电机的物理参数决定。

### **第 18 题**

**思路分析**
此题是关于一个由两台相同永磁直流电机组成的电动-发电机组。
* **第1问**: 写出电压平衡和转矩平衡方程。这是对基本公式的考察。
    * **电动机**: 电压平衡是 $U_{a1} = E_{a1} + I_{a1}R_a$。转矩平衡是 $T_{em1} = T + T_{f1}$，其中 $T$ 是传递给发电机的机械转矩，$T_{f1}$ 是电动机自身的摩擦转矩。
    * **发电机**: 电压平衡是 $E_{a2} = U_{a2} + I_{a2}R_a$，其中 $U_{a2} = I_{a2}R_L$。转矩平衡是 $T = T_{em2} + T_{f2}$，其中 $T_{em2}$ 是发电机的电磁转矩，$T_{f2}$ 是其摩擦转矩。
    * **系统**: 整个系统的转矩平衡为 $T_{em1} = T_{em2} + T_0$，其中 $T_0 = T_{f1} + T_{f2}$ 是整个机组的空载转矩。
* **第2问**: 在具体负载和电压下，求两台电机的电磁转矩和共同转速。
    1.  这是一个求解新稳态的问题，未知量是共同转速 $n$ (或角速度 $\omega$)。
    2.  需要先利用空载数据求出电机常数 $K_e$ (或 $K_T$)。空载时，电动机的电磁转矩 $T_{em0}$ 等于空载转矩 $T_{f0}$。
    3.  在新稳态下，可以分别表示出电动机和发电机的感应电动势 $E_{a1}, E_{a2}$ (都与 $n$ 相关)，进而表示出它们的电流 $I_{a1}, I_{a2}$ 和电磁转矩 $T_{em1}, T_{em2}$。
    4.  最后，利用系统的转矩平衡方程 $T_{em1} = T_{em2} + T_{f0}$ 建立一个只含未知数 $n$ 的方程，求解即可。
    5.  求出 $n$ 后，再代回之前建立的表达式，求出 $T_{em1}$ 和 $T_{em2}$。

**解答过程**

**1) 分别写出两台电机的电压平衡方程式和转矩平衡方程式。**
设电机转速为 $n$，电枢电阻为 $R_a$，电机常数为 $C_e, C_T$。
* **电动机 (左侧)**:
    * 电压平衡: $U_{a1} = E_{a1} + I_{a1} R_a = C_e n + I_{a1} R_a$
    * 转矩平衡: $T_{em1} = C_T I_{a1}$
* **发电机 (右侧)**:
    * 电压平衡: $E_{a2} = U_{a2} + I_{a2} R_a$。其中 $E_{a2} = C_e n$，负载电压 $U_{a2} = I_{a2} R_L$。所以 $C_e n = I_{a2} (R_L + R_a)$。
    * 转矩平衡: $T_{em2} = C_T I_{a2}$
* **整个机组**:
    * 转矩平衡: $T_{em1} = T_{em2} + T_0$ (其中 $T_0$ 是机组空载转矩)

**2) 当发电机负载电阻 $R_L=300\Omega$，电动机外加电压 $U_{a1}=110V$ 时，两台电机的电磁转矩各为多少？机组的共同转速为多少？**
1.  **求电机常数和空载转矩**:
    空载时 $T_0 = T_{f0} = 0.025 \, \text{N} \cdot \text{m}$，空载电流 $I_0 = 0.1 \, \text{A}$。
    根据 $T_0 = C_T I_0$，求得转矩常数：
    $C_T = \frac{T_0}{I_0} = \frac{0.025}{0.1} = 0.25 \, \text{N} \cdot \text{m/A}$
    由于是永磁电机，在国际单位制下 $K_e = K_T$，所以 $K_e = 0.25 \, \text{V/(rad/s)}$。

2.  **建立新稳态下的方程 (设新角速度为 $\omega$)**:
    * 电动机侧:
        $E_{a1} = K_e \omega = 0.25 \omega$
        $I_{a1} = \frac{U_{a1} - E_{a1}}{R_a} = \frac{110 - 0.25 \omega}{50}$
        $T_{em1} = K_T I_{a1} = 0.25 \times \frac{110 - 0.25 \omega}{50} = 0.005 (110 - 0.25 \omega)$
    * 发电机侧:
        $E_{a2} = K_e \omega = 0.25 \omega$
        $I_{a2} = \frac{E_{a2}}{R_a + R_L} = \frac{0.25 \omega}{50 + 300} = \frac{0.25 \omega}{350}$
        $T_{em2} = K_T I_{a2} = 0.25 \times \frac{0.25 \omega}{350} = \frac{0.0625 \omega}{350}$

3.  **建立并求解转矩平衡方程**:
    $T_{em1} = T_{em2} + T_0$
    $0.005 (110 - 0.25 \omega) = \frac{0.0625 \omega}{350} + 0.025$
    $0.55 - 0.00125 \omega = 0.0001786 \omega + 0.025$
    $0.525 = (0.00125 + 0.0001786) \omega$
    $0.525 = 0.0014286 \omega$
    $\omega = \frac{0.525}{0.0014286} \approx 367.5 \, \text{rad/s}$
    换算为转速 $n$:
    $n = \omega \times \frac{60}{2\pi} = 367.5 \times 9.55 \approx 3509 \, \text{r/min}$

4.  **计算电磁转矩**:
    $T_{em1} = 0.55 - 0.00125 \times 367.5 = 0.55 - 0.459 = 0.091 \, \text{N} \cdot \text{m}$
    $T_{em2} = \frac{0.0625 \times 367.5}{350} = \frac{22.97}{350} \approx 0.066 \, \text{N} \cdot \text{m}$
    (验证: $T_{em1} \approx 0.091$, $T_{em2} + T_0 \approx 0.066 + 0.025 = 0.091$，平衡成立)

**最终答案**:
1) 方程式已如上列出。
2) 电动机电磁转矩 $T_{em1} \approx \textbf{0.091 N·m}$，发电机电磁转矩 $T_{em2} \approx \textbf{0.066 N·m}$，机组共同转速约为 **3509 r/min**。

### **第 19 题**

**思路分析**
此题是关于一台直流伺服电动机在不同条件下的性能计算。
* **第1问**: 电枢电压 $U_a=18V$ 时，求启动转矩和空载转速。

    1.  **电枢电压**: 此时工作状态已知，可以直接用电压平衡方程 $U_a = K_e \omega + I_a R_a$ 求解。其中电流 $I_a = T_e / K_T$。
    2.  **机械转矩**: 机械转矩（输出到负载的转矩）$T_L = T_e - T_f$。

**解答过程**

1.  **求电机固有参数 $K_T, K_e, R_a$**:
    * 根据额定电磁转矩和额定电流求 $K_T$：
        $K_T = \frac{T_{emN}}{I_N} = \frac{0.015 \, \text{N} \cdot \text{m}}{0.5 \, \text{A}} = 0.03 \, \text{N} \cdot \text{m/A}$
    * 在国际单位制中，$K_e = K_T = 0.03 \, \text{V/(rad/s)}$。
    * 根据额定状态的电压平衡方程求 $R_a$：
        $U_N = K_e \omega_N + I_N R_a$
        $24 = 0.03 \times 300 + 0.5 \times R_a$

**1) 电枢电压 $U_a=18V$ 时**
* **启动转矩 $T_{st}$**:

    启动转矩 $T_{st} = K_T I_{st} = 0.03 \times 0.6 = 0.018 \, \text{N} \cdot \text{m}$
* **空载转速 $\omega_0$**:
    空载时电磁转矩等于空载摩擦转矩 $T_0 = T_f = 0.003 \, \text{N} \cdot \text{m}$。
    空载电流 $I_0 = \frac{T_0}{K_T} = \frac{0.003}{0.03} = 0.1 \, \text{A}$

    $18 = 0.03 \times \omega_0 + 0.1 \times 30$
    $18 = 0.03 \omega_0 + 3 \implies 0.03 \omega_0 = 15 \implies \omega_0 = 500 \, \text{rad/s}$

**2) 要求电磁转矩 $T_e=0.02 N·m$，转速为 $250 rad/s$**

    $U_a = K_e \omega + I_a R_a = 0.03 \times 250 + \frac{2}{3} \times 30 = 7.5 + 20 = 27.5 \, \text{V}$
* **电机输出的机械转矩 $T_L$ 是多少?**:
    机械转矩等于电磁转矩减去空载摩擦转矩：
    $T_L = T_e - T_f = 0.02 - 0.003 = 0.017 \, \text{N} \cdot \text{m}$

1) 启动转矩是 **0.018 N·m**，空载转速是 **500 rad/s**。
2) 要求电枢电压是 **27.5 V**，电机输出的机械转矩是 **0.017 N·m**。

好的，我们使用国际单位制（SI），将所有题目中的转速单位统一为 **rad/s**，并使用电机常数 **$K_e$ (V/(rad/s))** 和 **$K_T$ (N·m/A)** 进行分析和解答。在国际单位制下，对于同一台永磁或他励直流电机，这两个常数在数值上是相等的，即 $K_e = K_T$。

### **第 6 题 (使用 $K_e, K_T$)**

**思路分析**
1.  [cite_start]**目标**: 求解额定负载转矩 $T_L$ [cite: 3]。
2.  **核心公式**:
    * [cite_start]电压平衡: $U_a = E_a + I_a R_a$ [cite: 3]
    * 感应电动势: $E_a = K_e \omega$
    * 电磁转矩: $T_{em} = K_T I_a$
    * [cite_start]转矩平衡: $T_{em} = T_L + T_0$ [cite: 3]
3.  **解题步骤**:
    * 将额定转速 $n$ (r/min) 转换为角速度 $\omega$ (rad/s)。
    * 利用电压平衡方程求出额定工况下的感应电动势 $E_a$。
    * 利用 $E_a$ 和 $\omega$ 计算出电机的电动势常数 $K_e$。
    * 根据 $K_T = K_e$ 得到转矩常数。
    * 利用 $K_T$ 和额定电流计算电磁转矩 $T_{em}$。
    * 利用转矩平衡方程求出额定负载转矩 $T_L$。

**解答过程**
1.  **单位换算**:
    $\omega = n \times \frac{2\pi}{60} = 3600 \times \frac{2\pi}{60} = 120\pi \, \text{rad/s} \approx 377 \, \text{rad/s}$
2.  **求感应电动势 $E_a$**:
    [cite_start]$E_a = U_a - I_a R_a = 110 \, \text{V} - 0.4 \, \text{A} \times 50 \, \Omega = 90 \, \text{V}$ [cite: 3]
3.  **求电机常数 $K_e$ 和 $K_T$**:
    $K_e = \frac{E_a}{\omega} = \frac{90 \, \text{V}}{120\pi \, \text{rad/s}} \approx 0.2387 \, \text{V/(rad/s)}$
    $K_T = K_e \approx 0.2387 \, \text{N} \cdot \text{m/A}$
4.  **求电磁转矩 $T_{em}$**:
    $T_{em} = K_T I_a = 0.2387 \, \text{N} \cdot \text{m/A} \times 0.4 \, \text{A} \approx 0.0955 \, \text{N} \cdot \text{m}$
5.  **求额定负载转矩 $T_L$**:
    [cite_start]$T_L = T_{em} - T_0 = 0.0955 \, \text{N} \cdot \text{m} - 0.015 \, \text{N} \cdot \text{m} = 0.0805 \, \text{N} \cdot \text{m}$ [cite: 3]

**最终答案**: 该电机的额定负载转矩是 **0.0805 N·m**。

### **第 7 题 (使用 $K_e, K_T$)**

**思路分析**
[cite_start]这是一个电动-发电机组，两台电机完全相同且励磁相同，因此它们的 $K_e$ 和 $K_T$ 相等 [cite: 3]。

* [cite_start]**第1问**: 求解发电机空载电压 [cite: 3]。
    1.  将电动机的转速 $n$ 换算为角速度 $\omega$。
    2.  根据电动机的运行参数计算其感应电动势 $E_m$，进而求出电机常数 $K_e$。
    3.  发电机的空载感应电动势 $E_g$ 等于 $K_e \omega$。
    4.  空载时发电机端电压等于其感应电动势。
* [cite_start]**第2问**: 求解发电机带载后的机组转速 [cite: 3]。
    1.  设新的稳态角速度为 $\omega'$。
    2.  分别建立电动机和发电机的电压、电流、转矩与 $\omega'$ 之间的关系式。
    3.  建立整个系统的转矩平衡方程：电动机产生的电磁转矩 $T_{em,m}$ = 机组空载转矩 $T_0$ + 发电机产生的电磁负载转矩 $T_{em,g}$。
    4.  求解该方程得到新的角速度 $\omega'$。

**解答过程**

**1) 发电机空载时的电枢电压为多少？**
1.  **单位换算与求电机常数**:
    $\omega = 4500 \times \frac{2\pi}{60} = 150\pi \, \text{rad/s} \approx 471.2 \, \text{rad/s}$
    电动机感应电动势:
    [cite_start]$E_m = U_m - I_m R_a = 110 \, \text{V} - 0.12 \, \text{A} \times 75 \, \Omega = 101 \, \text{V}$ [cite: 3]
    电机常数:
    $K_e = \frac{E_m}{\omega} = \frac{101 \, \text{V}}{150\pi \, \text{rad/s}} \approx 0.2143 \, \text{V/(rad/s)}$
    $K_T = K_e \approx 0.2143 \, \text{N} \cdot \text{m/A}$
2.  **求发电机空载电压**:
    发电机空载感应电动势 $E_g = K_e \omega = 0.2143 \times 150\pi \approx 101 \, \text{V}$。
    空载时端电压 $U_{g0} = E_g = 101 \, \text{V}$。

**2) 发电机接上0.5kΩ的负载时，机组的转速是多少？**
1.  **求空载转矩 $T_0$**:
    $T_0$ 由初始空载电流产生：
    $T_0 = K_T I_m = 0.2143 \, \text{N} \cdot \text{m/A} \times 0.12 \, \text{A} \approx 0.0257 \, \text{N} \cdot \text{m}$
2.  **建立新稳态方程 (设新角速度为 $\omega'$)**:
    * 电动机电流: $I_m' = \frac{U_m - K_e \omega'}{R_a} = \frac{110 - 0.2143 \omega'}{75}$
    * 电动机转矩: $T_{em,m}' = K_T I_m' = 0.2143 \times \frac{110 - 0.2143 \omega'}{75}$
    * 发电机电流: $I_g' = \frac{E_g'}{R_a + R_L} = \frac{K_e \omega'}{75 + 500} = \frac{0.2143 \omega'}{575}$
    * 发电机转矩: $T_{em,g}' = K_T I_g' = 0.2143 \times \frac{0.2143 \omega'}{575}$
3.  **求解转矩平衡方程**:
    $T_{em,m}' = T_0 + T_{em,g}'$
    $0.2143 \times \frac{110 - 0.2143 \omega'}{75} = 0.0257 + 0.2143 \times \frac{0.2143 \omega'}{575}$
    两边同时除以 $K_T=0.2143$（近似认为$T_0$也是由$K_T$产生），等同于电流平衡：$I_m' = I_0 + I_g'$
    $\frac{110 - 0.2143 \omega'}{75} = 0.12 + \frac{0.2143 \omega'}{575}$
    $1.4667 - 0.002857 \omega' = 0.12 + 0.000373 \omega'$
    $1.3467 = 0.00323 \omega'$
    $\omega' = \frac{1.3467}{0.00323} \approx 416.9 \, \text{rad/s}$
4.  **换算为 r/min**:
    $n' = \omega' \times \frac{60}{2\pi} = 416.9 \times \frac{60}{2\pi} \approx 3981 \, \text{r/min}$

**最终答案**:
1) 发电机空载时的电枢电压为 **101 V**。
2) 机组的转速约为 **3981 r/min** (或 416.9 rad/s)。

### **第 15 题 (使用 $K_e, K_T$)**

**思路分析**
1.  [cite_start]**第1问**: 绘制机械特性 [cite: 1]。机械特性是 $\omega-T$ 关系曲线。需根据两个工作点数据联立求解电枢电阻 $R_a$ 和电机常数 $K_e$。
2.  [cite_start]**第2问**: 求机电时间常数 [cite: 1]。公式为 $\tau_m = \frac{J_m R_a}{K_e K_T}$。
3.  [cite_start]**第3问**: 讨论电压对时间常数的影响 [cite: 1]。分析 $\tau_m$ 公式中各参数与电压的关系。

**解答过程**
1.  **单位换算**:
    $\omega_1 = 4500 \times \frac{2\pi}{60} = 150\pi \, \text{rad/s} \approx 471.2 \, \text{rad/s}$
    $\omega_2 = 3000 \times \frac{2\pi}{60} = 100\pi \, \text{rad/s} \approx 314.2 \, \text{rad/s}$
2.  **求 $R_a$ 和 $K_e$**:
    * 点1: $100 = K_e \omega_1 + I_{a1} R_a = K_e(150\pi) + 0.05 R_a$
    * 点2: $100 = K_e \omega_2 + I_{a2} R_a = K_e(100\pi) + 0.2 R_a$
    联立求解：
    $K_e(150\pi) + 0.05 R_a = K_e(100\pi) + 0.2 R_a$
    $K_e(50\pi) = 0.15 R_a \implies R_a = \frac{50\pi}{0.15} K_e \approx 1047.2 K_e$
    代入第一个方程：
    $100 = K_e(150\pi) + 0.05 (1047.2 K_e) = 471.2 K_e + 52.36 K_e = 523.56 K_e$
    $K_e = \frac{100}{523.56} \approx 0.191 \, \text{V/(rad/s)}$
    $K_T = K_e \approx 0.191 \, \text{N} \cdot \text{m/A}$
    $R_a = 1047.2 \times 0.191 \approx 200 \, \Omega$
3.  **绘制机械特性**:
    理想空载角速度: $\omega_0 = \frac{U_a}{K_e} = \frac{100}{0.191} \approx 523.6 \, \text{rad/s}$ (即 5000 r/min)
    堵转转矩: $T_{st} = \frac{K_T U_a}{R_a} = \frac{0.191 \times 100}{200} = 0.0955 \, \text{N} \cdot \text{m}$
    在 $\omega-T$ 坐标系中，连接点 $(0, 523.6)$ 和 $(0.0955, 0)$ 即可。
4.  **求机电时间常数 $\tau_m$**:
    [cite_start]$\tau_m = \frac{J_m R_a}{K_e K_T} = \frac{(3.7 \times 10^{-6}) \times 200}{0.191 \times 0.191} \approx 0.0203 \, \text{s}$ [cite: 1]
5.  **讨论电压变化**:
    $\tau_m$ 的公式中各项均为电机固有物理参数，不随电枢电压变化。

**最终答案**:
1) 机械特性是一条直线，其理想空载角速度为 **523.6 rad/s** (5000 r/min)，堵转转矩为 **0.0955 N·m**。
2) 电机时间常数约为 **0.0203 s** (20.3 ms)。
3) 当电枢电压改变时，机电时间常数 **不变化**。

    [cite_start]$E_{a01} = 110 - 0.055 \times 80 = 105.6 \, \text{V}$ [cite: 1]
    $K_e = \frac{E_{a01}}{\omega_{01}} = \frac{105.6}{481.7} \approx 0.2192 \, \text{V/(rad/s)}$
    $K_T = K_e \approx 0.2192 \, \text{N} \cdot \text{m/A}$
**1) 电枢电压为70V时**
* 理想空载角速度 $\omega_{02}$:
    $\omega_{02} = \frac{U_{a2}}{K_e} = \frac{70}{0.2192} \approx 319.3 \, \text{rad/s}$ (约 3049 r/min)
* 堵转转矩 $T_{st}$:
    $T_{st} = \frac{K_T U_{a2}}{R_a} = \frac{0.2192 \times 70}{80} \approx 0.1918 \, \text{N} \cdot \text{m}$

**2) 驱动器供电时**
[cite_start]回路总电阻 $R_{total} = R_a + R_i = 80 + 50 = 130 \, \Omega$ [cite: 1]。
* 理想空载角速度 $\omega_{03}$: (空载电流为零，内阻无压降)
    $\omega_{03} = \frac{U_o}{K_e} = \frac{70}{0.2192} \approx 319.3 \, \text{rad/s}$ (约 3049 r/min)
* 启动转矩 $T_{st}'$:
    $T_{st}' = \frac{K_T U_o}{R_{total}} = \frac{0.2192 \times 70}{130} \approx 0.1179 \, \text{N} \cdot \text{m}$

**3) 负载转矩增加后的转速**
* **情况1 (直接70V供电)**:
    $\omega = \omega_{02} - \frac{R_a}{K_e K_T} T_c = 319.3 - \frac{80}{0.2192^2} \times 0.04 = 319.3 - 66.6 \approx 252.7 \, \text{rad/s}$ (约 2413 r/min)
* **情况2 (驱动器供电)**:
    $\omega' = \omega_{03} - \frac{R_{total}}{K_e K_T} T_c = 319.3 - \frac{130}{0.2192^2} \times 0.04 = 319.3 - 108.2 \approx 211.1 \, \text{rad/s}$ (约 2015 r/min)

**最终答案**:
1) 70V供电时，理想空载角速度约 **319.3 rad/s**，堵转电磁转矩约 **0.1918 N·m**。
2) 驱动器供电时，理想空载角速度约 **319.3 rad/s**，启动转矩约 **0.1179 N·m**。
3) 总负载增至0.04 N·m时，两种情况下的角速度分别约为 **252.7 rad/s** 和 **211.1 rad/s**。

### **第 18 题 (使用 $K_e, K_T$)**

**思路分析**
此题数据单位已基本为国际单位，分析过程与之前相同，重点在于使用角速度 $\omega$。
1.  [cite_start]**第1问**: 写出电压与转矩平衡方程 [cite: 2]。
2.  [cite_start]**第2问**: 求解负载下的电磁转矩和共同角速度 [cite: 2]。
    * [cite_start]根据空载数据 $T_0=0.025 \text{N·m}$ 和 $I_0=0.1 \text{A}$ 求出 $K_T$，并令 $K_e=K_T$ [cite: 2]。
    * [cite_start]建立系统转矩平衡方程 $T_{em1} = T_{em2} + T_0$，其中各项都表示为角速度 $\omega$ 的函数 [cite: 2]。
    * 求解方程得到 $\omega$，再反代入求得两个电磁转矩。

**解答过程**

**1) 方程式**
* 电动机: $U_{a1} = K_e \omega + I_{a1} R_a$; $T_{em1} = K_T I_{a1}$
* 发电机: $K_e \omega = I_{a2}(R_L + R_a)$; $T_{em2} = K_T I_{a2}$
* 机组: $T_{em1} = T_{em2} + T_0$

**2) 求解稳态工作点**
1.  **求电机常数**:
    [cite_start]$K_T = \frac{T_0}{I_0} = \frac{0.025 \, \text{N} \cdot \text{m}}{0.1 \, \text{A}} = 0.25 \, \text{N} \cdot \text{m/A}$ [cite: 2]
    $K_e = K_T = 0.25 \, \text{V/(rad/s)}$
2.  **建立并求解转矩平衡方程**:
    $T_{em1} = K_T \frac{U_{a1} - K_e \omega}{R_a} = 0.25 \frac{110 - 0.25 \omega}{50}$
    $T_{em2} = K_T \frac{K_e \omega}{R_a + R_L} = 0.25 \frac{0.25 \omega}{50 + 300}$
    $0.25 \frac{110 - 0.25 \omega}{50} = 0.25 \frac{0.25 \omega}{350} + 0.025$
    两边除以 0.25:
    $\frac{110 - 0.25 \omega}{50} = \frac{0.25 \omega}{350} + 0.1$
    $2.2 - 0.005 \omega = 0.000714 \omega + 0.1$
    $2.1 = 0.005714 \omega$
    $\omega = \frac{2.1}{0.005714} \approx 367.5 \, \text{rad/s}$
3.  **计算电磁转矩**:
    $T_{em1} = 0.25 \frac{110 - 0.25(367.5)}{50} \approx 0.091 \, \text{N} \cdot \text{m}$
    $T_{em2} = 0.25 \frac{0.25(367.5)}{350} \approx 0.066 \, \text{N} \cdot \text{m}$

**最终答案**:
1) 方程式已如上列出。
2) 电动机电磁转矩 $T_{em1} \approx \textbf{0.091 N·m}$，发电机电磁转矩 $T_{em2} \approx \textbf{0.066 N·m}$，机组共同角速度约为 **367.5 rad/s**。

### **第 19 题 (使用 $K_e, K_T$)**

**思路分析**
此题数据单位已为国际单位，解答过程不变。
1.  [cite_start]**第1问**: 求18V电压下的启动转矩和空载角速度 [cite: 2]。
    * 先根据额定数据求出电机参数 $K_T, K_e, R_a$。
    * 启动时 $\omega=0$，空载时 $T_{em} = T_f$。
2.  [cite_start]**第2问**: 计算特定工况下的电压和输出转矩 [cite: 2]。
    * 根据 $T_e = K_T I_a$ 求电流，再用电压平衡方程求电压。
    * 输出机械转矩 $T_L = T_e - T_f$。

**解答过程**

1.  **求电机固有参数**:
    [cite_start]$K_T = \frac{T_{emN}}{I_N} = \frac{0.015}{0.5} = 0.03 \, \text{N} \cdot \text{m/A}$ [cite: 2]
    $K_e = K_T = 0.03 \, \text{V/(rad/s)}$
    [cite_start]$R_a = \frac{U_N - K_e \omega_N}{I_N} = \frac{24 - 0.03 \times 300}{0.5} = \frac{24 - 9}{0.5} = 30 \, \Omega$ [cite: 2]

**1) 电枢电压 $U_a=18V$ 时**
* 启动转矩 $T_{st}$:
    $T_{st} = \frac{K_T U_a}{R_a} = \frac{0.03 \times 18}{30} = 0.018 \, \text{N} \cdot \text{m}$
* 空载角速度 $\omega_0$:
    [cite_start]空载电流 $I_0 = \frac{T_f}{K_T} = \frac{0.003}{0.03} = 0.1 \, \text{A}$ [cite: 2]
    $\omega_0 = \frac{U_a - I_0 R_a}{K_e} = \frac{18 - 0.1 \times 30}{0.03} = \frac{18 - 3}{0.03} = 500 \, \text{rad/s}$

**2) 要求 $T_e=0.02 N·m$，$\omega = 250 rad/s$**
* 电枢电压 $U_a$:
    电流 $I_a = \frac{T_e}{K_T} = \frac{0.02}{0.03} = \frac{2}{3} \, \text{A}$
    $U_a = K_e \omega + I_a R_a = 0.03 \times 250 + \frac{2}{3} \times 30 = 7.5 + 20 = 27.5 \, \text{V}$
* 输出机械转矩 $T_L$:
    [cite_start]$T_L = T_e - T_f = 0.02 - 0.003 = 0.017 \, \text{N} \cdot \text{m}$ [cite: 2]

**最终答案**:
1) 启动转矩是 **0.018 N·m**，空载转速是 **500 rad/s**。
2) 要求电枢电压是 **27.5 V**，电机输出的机械转矩是 **0.017 N·m**。
