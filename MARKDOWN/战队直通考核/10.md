好的，我们来证明这个关于矩阵指数的重要性质。

**所要证明的性质是：**
如果两个 $n \times n$ 的方阵 $A$ 和 $F$ 满足交换律，即 $AF = FA$，那么对于任意实数 $t$，有：
$$e^{(A+F)t} = e^{At} e^{Ft}$$

同时，我们也要说明为什么当 $AF \neq FA$ 时，此式不成立。

---

### 证明过程

证明的核心是利用矩阵指数的泰勒级数定义，并结合二项式定理。

**1. 写出矩阵指数的定义**

对于任意一个方阵 $M$，其矩阵指数 $e^M$ 定义为以下幂级数：
$$e^M = I + M + \frac{M^2}{2!} + \frac{M^3}{3!} + \dots = \sum_{k=0}^{\infty} \frac{M^k}{k!}$$
其中 $I$ 是单位矩阵，$0! = 1$。

将这个定义应用到我们要证明的等式的左边和右边：

* **等式左边**:
    $$e^{(A+F)t} = e^{t(A+F)} = \sum_{k=0}^{\infty} \frac{[t(A+F)]^k}{k!} = \sum_{k=0}^{\infty} \frac{t^k(A+F)^k}{k!}$$

* **等式右边**:
    $$e^{At} e^{Ft} = \left( \sum_{i=0}^{\infty} \frac{(At)^i}{i!} \right) \left( \sum_{j=0}^{\infty} \frac{(Ft)^j}{j!} \right) = \left( \sum_{i=0}^{\infty} \frac{t^i A^i}{i!} \right) \left( \sum_{j=0}^{\infty} \frac{t^j F^j}{j!} \right)$$

我们的目标是证明这两个表达式相等。通常，从较复杂的一侧（右侧）开始，将其化简成左侧的形式会更容易。

**2. 展开并整理等式右边**

我们将右边的两个无穷级数相乘。这被称为柯西乘积（Cauchy Product）：
$$e^{At} e^{Ft} = \sum_{i=0}^{\infty} \sum_{j=0}^{\infty} \frac{t^{i+j} A^i F^j}{i! j!}$$
为了与左边的形式进行比较，我们希望将这个双重求和变成一个单一的求和。我们令 $k = i+j$，然后对 $k$ 从 $0$ 到 $\infty$ 进行求和。对于每一个固定的 $k$， $i$ 可以从 $0$ 取到 $k$ (此时 $j = k-i$)。
这样，求和形式变为：
$$e^{At} e^{Ft} = \sum_{k=0}^{\infty} \sum_{i=0}^{k} \frac{t^k A^i F^{k-i}}{i! (k-i)!}$$我们将与内层求和无关的项 $t^k$ 提出来，并为了凑出二项式系数，在分子分母同乘以 $k!$:$$e^{At} e^{Ft} = \sum_{k=0}^{\infty} \frac{t^k}{k!} \left( \sum_{i=0}^{k} \frac{k!}{i! (k-i)!} A^i F^{k-i} \right)$$

**3. 应用交换律条件 ($AF=FA$)**

观察括号内的表达式：
$$\sum_{i=0}^{k} \frac{k!}{i! (k-i)!} A^i F^{k-i} = \sum_{i=0}^{k} \binom{k}{i} A^i F^{k-i}$$
这**非常像**二项式展开 $(A+F)^k$。
**然而，矩阵的二项式定理 $(A+F)^k = \sum_{i=0}^{k} \binom{k}{i} A^i F^{k-i}$ 能够成立的前提条件，正是 $A$ 和 $F$ 可以交换，即 $AF=FA$。**

为什么呢？我们来看一个简单的例子，$k=2$：
$$(A+F)^2 = (A+F)(A+F) = A(A+F) + F(A+F) = A^2 + AF + FA + F^2$$只有当 $AF=FA$ 时，中间的两项才能合并，得到：$$(A+F)^2 = A^2 + 2AF + F^2 = \binom{2}{0}F^2 + \binom{2}{1}AF + \binom{2}{2}A^2$$
这正好符合二项式展开的形式。如果 $AF \neq FA$，则 $(A+F)^2 \neq A^2 + 2AF + F^2$，二项式定理不成立。

回到我们的证明，因为题目给定了条件 $AF=FA$，所以矩阵的二项式定理成立。于是：
$$\sum_{i=0}^{k} \binom{k}{i} A^i F^{k-i} = (A+F)^k$$

**4. 完成证明**

将上一步的结果代入第 2 步的表达式中：
$$e^{At} e^{Ft} = \sum_{k=0}^{\infty} \frac{t^k}{k!} (A+F)^k = \sum_{k=0}^{\infty} \frac{[t(A+F)]^k}{k!}$$
这个结果正好就是 $e^{(A+F)t}$ 的定义式。
因此，我们证明了：
$$e^{(A+F)t} = e^{At} e^{Ft}$$
**证毕。**

---

### 结论与注解

* **证明的关键**：在于将 $e^{At}e^{Ft}$ 的级数乘积，通过变量代换($k=i+j$)和重新组合，整理成二项式求和的形式。
* **交换律的必要性**：交换律 $AF=FA$ 是应用矩阵二项式定理 $(A+F)^k = \sum \binom{k}{i} A^i F^{k-i}$ 的**充要条件**。如果没有这个条件，级数乘积无法被合并成 $(A+F)^k$ 的形式，整个推导在第 3 步就中断了。因此，如果 $A, F$ 不可交换，该性质**不成立**。这是矩阵代数区别于标量代数的一个核心要点。
